Introduction

Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible.

It is a general-purpose programming language intended to let application developers write once, run anywhere (WORA), meaning that compiled Java code can run on all platforms that support Java without the need for recompilation.

Java applications are typically compiled to bytecode that can run on any Java virtual machine (JVM) regardless of the underlying computer architecture.

The syntax of Java is similar to C and C++, but it has fewer low-level facilities than either of them.

The Java runtime provides dynamic capabilities (such as reflection and runtime code modification) that are typically not available in traditional compiled languages.

System.out.println("Hello, World!");
  • Platform independent
  • Object-oriented
  • Robust and secure
  • Multithreaded
  • High performance
Syntax

Java syntax is largely influenced by C++. This section covers the basic syntax of Java.

A basic Java program consists of a class definition with a main method.

public class Main {
 public static void main(String[] args) {
  System.out.println("Hello, World!");
 }
}

Every statement in Java must end with a semicolon (;).

Blocks of code are defined using curly braces { and }.

Java is case-sensitive, meaning that "Main" and "main" are different identifiers.

Java classes are defined using the "class" keyword.

Variables

Variables in Java are used to store data for processing. A variable is a container that holds data that can be changed during the execution of a program.

Java has three types of variables: local, instance, and static.

int number = 5;
String name = "John";
double price = 19.99;

Variables must be declared before they are used.

Java is a strongly-typed language, meaning that each variable must be declared with a data type.

Variable names are case-sensitive and must begin with a letter, underscore (_), or dollar sign ($).

Data Types

Java supports a variety of data types, which can be categorized into two types: primitive and non-primitive.

Primitive data types include byte, short, int, long, float, double, boolean, and char.

Non-primitive data types include Classes, Interfaces, and Arrays.

int age = 30;
char grade = 'A';
boolean isStudent = false;

Each data type has a specific size and range. For example, an int is a 32-bit signed integer, whereas a byte is an 8-bit signed integer.

Non-primitive data types are created by the programmer and are not predefined by Java.

Control Flow

Control flow statements in Java allow you to control the order in which statements are executed.

Common control flow statements include if, else, switch, for, while, and do-while.

if (x > 0) {
 System.out.println("x is positive");
} else {
 System.out.println("x is non-positive");
}

The if statement executes a block of code if its condition is true.

The else statement executes a block of code if the condition in the if statement is false.

Switch statements allow for the selection of one of many blocks of code to execute.
